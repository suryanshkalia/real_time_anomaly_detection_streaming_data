# Real-Time Anomaly Detection in Streaming Data

## Overview
This project simulates a real-time data streaming system using an asynchronous pipeline. The goal is to process continuous data and detect anomalies.

---

## Architecture
Input → Anomaly Detection → Output

- Input: Generates streaming numeric data
- Anomaly Detection: Identifies abnormal values using threshold logic
- Output: Displays processed data

---

## Features
- Async pipeline using Python asyncio
- Rate-controlled streaming (real-time simulation)
- Basic anomaly detection
